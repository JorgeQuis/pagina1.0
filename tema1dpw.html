INTRODUCCIÓN - CONCEPTOS CLAVE



PÁGINA WEB

Una página web es un documento digital que contiene información y es accesible a través de Internet mediante un navegador web.



SITIO WEB

Un sitio web es un conjunto de páginas web interconectadas y organizadas bajo un mismo dominio.



SERVIDOR WEB

Un servidor web es una computadora o sistema de software diseñado para almacenar, procesar y entregar páginas web.



MOTOR DE BUSQUEDA

Un motor de busqueda es un programa que actua como una herramienta en línea que permite a los usuarios encontrar información en Internet.



NAVEGADOR WEB

Un navegador web es un programa o aplicación de software que permite a los usuarios acceder, visualizar y navegar por contenido en Internet.



Una página web esta compuesta de diferentes capas. Estas son las capas fundamentales del desarrollo front-end.
